# Valet Character Images

Put Valet mascot images in this directory for use in Mintlify docs.

Mintlify will be configured with `/docs` as the monorepo documentation path, so files here are served at:

```text
/images/characters/<filename>
```

Current pose names:

- `wave.png`
- `pointing-right.png`
- `thinking.png`
- `writing.png`
- `thumbs-up.png`
- `stop.png`
- `no.png`
- `laptop.png`
- `laying.png`
- `workspace.png`
- `child-sessions.png`
- `personas.png`

Use one mascot per page or major callout. Do not recolor, stretch, rotate, or place the mascot over busy screenshots or live text. Include meaningful alt text in MDX pages that reference an image.
