---
title: Channels
description: Work with Valet from the web app, Slack, Telegram, and other connected places.
---

Channels are the places where people talk to Valet. A task can start in one place and return updates in the same place.

## Web app

Use the web app when you want the clearest view of the work.

The web app is best for:

- Starting a new task.
- Watching progress.
- Reviewing approvals.
- Inspecting files, drafts, and results.
- Managing settings.

## Slack

Use Slack for quick requests and updates.

Slack is best for:

- Asking Valet to check something.
- Getting a status update.
- Receiving approval prompts.
- Posting summaries after workflows.

## Telegram

Use Telegram when you want a lightweight mobile channel for personal tasks, quick updates, or approvals.

## Choosing the right channel

| Use this | When |
| --- | --- |
| Web app | The task is new, sensitive, or needs close review. |
| Slack | The request belongs in team chat or needs a quick follow-up. |
| Telegram | You want a personal mobile place to work with Valet. |
| Workflow | The work should run on a schedule or after a repeated event. |

## Channel safety

Channels can include sensitive information. Admins should decide:

- Which channels are enabled.
- Which channels can start tasks.
- Which channels can show approval prompts.
- Which channels can reach personal or team memory.
