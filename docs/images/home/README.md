# Homepage Assets

Use this folder for homepage-specific art.

Expected files:

- `hero-light.*` and `hero-dark.*` for the main homepage illustration.
- `grid-light.svg` and `grid-dark.svg` if the homepage needs an exported grid rather than CSS.
- Section illustrations that are only used on the custom docs homepage.

The current homepage uses CSS and existing character art as placeholders. Replace those references when the final brand kit arrives.
