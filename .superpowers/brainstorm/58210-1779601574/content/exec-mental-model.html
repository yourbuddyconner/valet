A · Timeline cards
Vertical timeline. Each step is a card with inline output preview. Loops indent their iterations as nested cards. CI / GitHub Actions feel.
B · Diagram-focus split
Diagram is the navigation. Clicking a node focuses its detail panel. Selected step shows input + output. Architectural / "I think in graphs" feel.
C · Report (collapsed)
Linear list, everything collapsed by default. Click a row to expand its IO. Compact, scan-friendly. Build log / test report feel.
