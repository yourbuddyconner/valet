## Setup

No authentication — this integration calls a public service and does not require credentials.

## Upstream documentation

This integration proxies the Turnkey Docs MCP server. For the full, current list of available actions, see the [Turnkey documentation](https://docs.turnkey.com).

## Actions

Actions for Turnkey Docs are proxied through a remote [Model Context Protocol](https://modelcontextprotocol.io) server at `https://docs.turnkey.com/mcp`. The exact list of actions, their parameters, and behavior are owned by the upstream MCP server — Valet enumerates them at runtime when the integration is connected.

Default risk level applied to Turnkey Docs actions: `low`. Individual actions may carry their own risk level from the upstream server.

Use `list_tools` with `service: turnkey_docs` from an agent session to see the live action list, parameter schemas, and per-action risk levels.
