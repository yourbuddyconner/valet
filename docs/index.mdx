# Valet

Valet is a calm, confident AI coworker with its own hosted workstation. It can work in the background, use connected tools, remember preferences, coordinate tasks, ask for approvals, and meet users where they already work.

For engineering work, that workstation includes a repository checkout, VS Code, a browser, a terminal, and an agent runtime. For broader operational work, the same model gives Valet a place to use integrations, follow workflows, handle channel messages, and keep durable context across tasks.

## Product model

Valet has three core layers:

- **Sessions** are the units of agent work. A session owns the live workspace for a task, including message history, runtime state, participants, access controls, and any technical sandbox state.
- **Orchestrators** are persistent coordinator agents. Each user gets an orchestrator that can answer questions, remember preferences, route channel messages, manage task context, and spawn child sessions.
- **Plugins** provide integrations, channels, skills, personas, and sandbox tools. Some plugins expose first-party actions, while others proxy remote MCP services.
