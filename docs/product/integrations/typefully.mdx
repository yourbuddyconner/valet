## Setup

API key — the user pastes an API key issued by the provider during connection setup.



## Actions

Actions for Typefully are proxied through a remote [Model Context Protocol](https://modelcontextprotocol.io) server at `https://mcp.typefully.com/mcp`. The exact list of actions, their parameters, and behavior are owned by the upstream MCP server — Valet enumerates them at runtime when the integration is connected.

Default risk level applied to Typefully actions: `medium`. Individual actions may carry their own risk level from the upstream server.

Use `list_tools` with `service: typefully` from an agent session to see the live action list, parameter schemas, and per-action risk levels.
