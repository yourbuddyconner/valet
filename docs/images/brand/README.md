# Brand Assets

Drop the final Valet brand kit here when it is ready.

Expected files:

- `logo-light.svg` and `logo-dark.svg` if the current logo files need replacement.
- `favicon.svg` if the favicon changes.
- `og-image.png` for social previews.
- `palette.png` or another visual reference for the approved color system.
- `type-specimen.png` or font notes if the docs need custom font handling.

Keep filenames lowercase and hyphenated so they are easy to reference from MDX and CSS.
